# Fabric Sales Analytics Project

Built an end-to-end analytics solution using Microsoft Fabric.

## Architecture
CSV → Lakehouse → Fact & Dimension Tables → Semantic Model → Power BI Dashboard

## Tools Used
- Microsoft Fabric
- PySpark
- Lakehouse
- Power BI Semantic Model
- DAX Measures

## Features
- Star schema modeling
- Data type standardization
- Relationship creation
- KPI dashboard
- Scheduled refresh setup
